# Быстрая настройка Railway

## Шаг 1: Добавьте Redis сервис

1. Откройте https://railway.app
2. Выберите ваш проект
3. Нажмите **"+ New"** → **"Database"** → **"Add Redis"**
4. Railway создаст Redis сервис автоматически

## Шаг 2: Настройте переменные окружения для бота

1. Откройте сервис **бот** (не Redis) в вашем проекте
2. Перейдите в раздел **"Variables"**
3. Нажмите **"+ New Variable"** и добавьте следующие переменные:

### Обязательные переменные:

| Переменная | Значение |
|------------|----------|
| `TELEGRAM_BOT_TOKEN` | `8074208505:AAEvJ0vqw6TVGs06gOWqxf2h8B1MjWnnBWI` |
| `TELEGRAM_ADMINS` | `nikiname` |
| `REDIS_DB` | `0` |
| `TRIGGER_WORDS` | `жопа,мат,слово1,слово2` |

### Redis переменные (из Redis сервиса):

1. Откройте **Redis сервис** в вашем проекте
2. Перейдите в **"Variables"**
3. Найдите и скопируйте значения:
   - `REDIS_HOST`
   - `REDIS_PORT`
   - `REDIS_PASSWORD`

4. Вернитесь в сервис **бот** → **"Variables"**
5. Добавьте скопированные переменные:
   - `REDIS_HOST` = (значение из Redis)
   - `REDIS_PORT` = (значение из Redis)
   - `REDIS_PASSWORD` = (значение из Redis)

## Шаг 3: Деплой

После добавления всех переменных Railway автоматически начнет деплой.

Проверьте логи:
- Откройте сервис бота → **"Deployments"** → выберите последний деплой → **"View Logs"**

Должно появиться: `Bot started and ready!`

## Шаг 4: Проверка работы

Отправьте боту в Telegram команду `/start` - он должен ответить справкой.

## Troubleshooting

**Бот не запускается:**
- Проверьте, что все переменные установлены
- Проверьте логи на наличие ошибок
- Убедитесь, что Redis сервис запущен

**Ошибка подключения к Redis:**
- Убедитесь, что Redis сервис добавлен в проект
- Проверьте, что переменные REDIS_HOST, REDIS_PORT, REDIS_PASSWORD установлены правильно
- Используйте значения из Redis сервиса Railway (не localhost!)
